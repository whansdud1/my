## Core Skills

- **Frontend Design**: Vue/React 컴포넌트 설계 및 스타일링, UI 구현 전문
  - 예시: `/frontend-design "로그인 폼 컴포넌트 구현"`

## Plugin Skills

- **/specify**: 기능 명세서(spec.md) 작성 및 관리
  - 예시: `/specify "사용자 인증 기능"`
- **/plan**: 구현 계획(plan.md) 수립 및 설계
  - 예시: `/plan`
- **/tasks**: 작업 목록(tasks.md) 생성 및 관리
  - 예시: `/tasks`
- **/implement**: 계획에 따른 코드 구현 실행
  - 예시: `/implement`
