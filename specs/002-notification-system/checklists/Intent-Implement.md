## Core Skills

- **Frontend Design**: Vue/React 컴포넌트 설계 및 스타일링, UI 구현 전문
  - 예시: `/frontend-design "로그인 폼 컴포넌트 구현"`
- **Component Architecture**: 재사용 가능한 컴포넌트 구조 설계 및 패턴 적용
  - 예시: `/component-architecture "사용자 프로필 카드 컴포넌트"`
- **Responsive Layout**: 반응형 레이아웃 설계 및 미디어 쿼리 구현
  - 예시: `/responsive-layout "대시보드 레이아웃 반응형 구현"`
- **Testing**: 단위 테스트, 통합 테스트, E2E 테스트 작성
  - 예시: `/testing "로그인 기능 테스트 코드"`
- **CI/CD Pipeline**: GitHub Actions, GitLab CI 파이프라인 구성
  - 예시: `/cicd "GitLab CI 파이프라인 설정"`
- **Network Inspector**: API 요청/응답 분석 및 네트워크 디버깅
  - 예시: `DevTools > Network 탭 활용`

## Plugin Skills

- **/specify**: 기능 명세서(spec.md) 작성 및 관리
  - 예시: `/specify "사용자 인증 기능"`
- **/plan**: 구현 계획(plan.md) 수립 및 설계
  - 예시: `/plan`
- **/tasks**: 작업 목록(tasks.md) 생성 및 관리
  - 예시: `/tasks`
- **/implement**: 계획에 따른 코드 구현 실행
  - 예시: `/implement`
- **Ralph Wiggum Debug**: 디버깅 및 트러블슈팅 전문 에이전트
  - 예시: `/debug "API 500 에러 원인 분석"`

## MCP Plugins

- **Context7 Docs**: 라이브러리 최신 문서 실시간 조회 (MCP)
  - 예시: `mcp__context7__resolve_library_id → mcp__context7__get_library_docs`
